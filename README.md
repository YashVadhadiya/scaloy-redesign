# Scaloy Technology Website Redesign

A modern, premium, and award-level website for Scaloy Technology - an IT services and consulting company specializing in web development, mobile apps, AI/ML, and blockchain solutions.

## Overview

This project includes a complete redesign of the Scaloy Technology website with:
- **Modern Dark Theme** with gradient accents
- **Premium UI/UX** with glassmorphism effects
- **Smooth Animations** and micro-interactions
- **Fully Responsive** design for all devices

## Website Structure

### Pages Created:
1. **index.html** - Homepage with hero, services, why choose us, projects, testimonials, and CTA
2. **about.html** - Company story, timeline, and team
3. **services.html** - Detailed service offerings with process
4. **portfolio.html** - Project showcase with filtering
5. **contact.html** - Contact form and information

## Design System

### Colors
- Primary Dark: `#0a0a0f`
- Primary Accent: `#00d4ff` (Electric Cyan)
- Secondary Accent: `#7b2cbf` (Royal Purple)
- Surface Dark: `#12121a`
- Surface Light: `#1a1a25`

### Typography
- Headings: Space Grotesk
- Body: DM Sans

### Features
- Animated counters
- Scroll reveal animations
- Glassmorphism cards
- Gradient borders and glows
- Responsive navigation
- Filterable portfolio

## Tech Stack
- HTML5
- CSS3 (Custom properties, Grid, Flexbox)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts

## How to Use

Simply open `index.html` in a web browser to view the website. All pages are linked and fully functional.
